# 📺 FitStream —— Your Immersive Workout Video Companion

[![React](https://img.shields.io/badge/React-19.0-blue.svg)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Vite-6.2-646CFF.svg)](https://vitejs.dev/)
[![TailwindCSS](https://img.shields.io/badge/Tailwind-4.0-38B2AC.svg)](https://tailwindcss.com/)

**Make "Watch & Workout" simpler and more focused.**

FitStream is a productivity tool designed for "micro-workout" fans. It integrates a **YouTube video window** with a **workout timer** into a single interface.

**💡 Core Concept:**
Reminds you to stay active while watching videos or binging shows. Stretch your neck, do facial exercises, or perform quick routines during your screen time.

---

## ✨ Key Features

- **Split Layout**: Immersive YouTube playback on the left, real-time timer and exercise prompts on the right.
- **Smart YouTube Integration**: Easily load any YouTube video by pasting the URL.
- **Customizable Exercise Sequences**:
  - **Full Control**: Click the **⚙️ Gear Icon** to add, remove, or rename exercises.
  - **Precise Timing**: Set independent durations for each movement.
  - **Auto-Flow**: The timer automatically switches to the next exercise when finished.
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop screens.
- **Visual Progress**: Dynamic progress ring with large, clear countdown.

---

## 📖 How to Use

1. **Load a Video**: Paste a YouTube link in the top input and click "Load".
2. **Start Timer**: Click the large play button on the right to start your routine.
3. **Customize**: Click the gear icon to modify exercise names (e.g., Neck Stretch, Face Yoga) and durations.

## 🚀 Deployment

If you want to host your own version of FitStream:

### 1. One-Click Deploy
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fyour-username%2Ffitstream)

### 2. Local Setup
Ensure [Node.js](https://nodejs.org/) is installed.

```bash
# Clone the repo
git clone https://github.com/your-username/fitstream.git

# Install dependencies
npm install

# Start development server
npm run dev
```
